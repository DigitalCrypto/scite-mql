SciTE Customization GUI

This tool allows you to change the theme of your Scite editor quickly and seemlessly.

Simply download the file extract it and run it.

There are a number of builtin themes already.  You can make any changes to pre-existing themes or you can build a new one from scratch.

This program was found at:
http://www.autoitscript.com/forum/topic/138501-scite-customization-gui-02262015-a/

You can get more information from this forum on this software.
